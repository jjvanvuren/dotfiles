# Hyprland Keybinding Cheatsheet

> `SUPER` = Windows/Meta key

## Applications

| Binding | Action |
|---------|--------|
| `SUPER + Return` | Terminal (kitty) |
| `SUPER + SHIFT + Return` | Browser (zen-browser) |
| `SUPER + Space` | App launcher (wofi) |
| `SUPER + E` | File manager (thunar) |

## Session

| Binding | Action |
|---------|--------|
| `SUPER + Escape` | Lock screen |
| `SUPER + SHIFT + Z` | Lock & suspend |
| `SUPER + SHIFT + Escape` | Shutdown |
| `SUPER + SHIFT + C` | Reload config |

## Windows

| Binding | Action |
|---------|--------|
| `SUPER + Q` | Close window |
| `SUPER + V` | Toggle floating |
| `SUPER + F` | Fullscreen (hides bar) |
| `SUPER + SHIFT + F` | Maximize (keeps bar) |
| `SUPER + P` | Pseudotile |
| `SUPER + T` | Toggle split direction |
| `SUPER + A` | Cycle floating/tiling focus |
| `SUPER + SHIFT + V` | Preselect horizontal split |
| `SUPER + SHIFT + H` | Preselect vertical split |

| Binding | Action |

## Resize Mode

| Binding | Action |
|---------|--------|
| `SUPER + R` | Enter resize mode |
| `H / J / K / L` | Resize (10px) |
| `SHIFT + H / J / K / L` | Resize (50px) |
| `Escape` or `SUPER + R` | Exit resize mode |

## Workspaces

| Binding | Action |
|---------|--------|
| `SUPER + 1-0` | Switch to workspace 1-10 |
| `SUPER + SHIFT + 1-0` | Move window to workspace 1-10 |
| `SUPER + Scroll` | Cycle workspaces |
| `SUPER + S` | Toggle scratchpad |
| `SUPER + SHIFT + S` | Move to scratchpad |

## Window Groups (Tabs)

| Binding | Action |
|---------|--------|
| `SUPER + G` | Toggle group |
| `SUPER + Tab` | Next tab |
| `SUPER + Backspace` | Previous tab |
| `SUPER + SHIFT + G` | Move into group |
| `SUPER + SHIFT + E` | Move out of group |

## Mouse

| Binding | Action |
|---------|--------|
| `SUPER + LMB drag` | Move window |
| `SUPER + RMB drag` | Resize window |
| `SUPER + Scroll` | Switch workspace |

## Screenshots

| Binding | Action |
|---------|--------|
| `Print` | Select region |
| `SHIFT + Print` | Current monitor |
| `CTRL + Print` | Active window |

## Clipboard

| Binding | Action |
|---------|--------|
| `SUPER + SHIFT + V` | Clipboard history |

## Notifications

| Binding | Action |
|---------|--------|
| `SUPER + N` | Show last notification |
| `SUPER + SHIFT + N` | Clear all notifications |

## Bar

| Binding | Action |
|---------|--------|
| `SUPER + SHIFT + B` | Toggle waybar |
